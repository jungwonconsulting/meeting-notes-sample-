# 부동산 매물 판단 시스템 사용법

## 시스템 개요

부동산 매물별로 조사 정보를 축적하고, AI를 활용해 구조화된 분석과 매수 판단 문서를 생성하는 시스템입니다.
개별 매물 분석(미시)과 시장환경 분석(거시)을 병행하여 매입 결정을 지원합니다.

## 폴더 구조

```
부동산전략/
├── 00_사용법/          : 본 안내 문서
├── 01_시스템/
│   ├── 템플릿/         : AI 실패시 수동 작성용 양식
│   └── 프롬프트/       : AI 생성 지시 프롬프트
├── 10_매물/            : 매물별 조사 및 분석 데이터
├── 20_비교/            : 매물간 비교 분석 문서
└── 30_시장환경/        : 거시 환경 (정책, 금리, 규제 등)
```

---

## 매물 등록 방법

1. `10_매물/` 아래에 `M[번호]_[매물명]` 폴더 생성 (예: `M00001_강남역오피스텔`)
2. `00_매물_기본정보.md`를 템플릿 기준으로 작성
3. `01_조사/` 폴더 생성

## 조사 추가 방법

1. `01_조사/` 아래에 `YYYY-MM-DD_[조사유형]` 폴더 생성
   - 조사유형 예시: 시세조사, 현장방문, 중개사통화, 뉴스정책, 법률검토, 세금검토, 임대수익조사, 기타
2. `01_원본.md`에 원본 정보 붙여넣기 (이후 수정 불가)
3. AI에게 정리 요청

## 시장환경 정보 추가 방법

1. `30_시장환경/01_원본/` 아래에 `YYYY-MM-DD_[주제]` 폴더 생성
2. `01_원본.md`에 뉴스, 정책발표, 통계 등 원본 붙여넣기
3. AI에게 시장환경 정리 요청

---

## 처리 단계

### 미시 파이프라인 (매물별)

| 단계 | 입력 | 출력 | 사용 프롬프트 |
|------|------|------|--------------|
| 1 | 01_원본.md | 02_정리.md | 01_정보정리_프롬프트.md |
| 2 | 전체 02_정리.md들 | 02_조사분석.md | 02_조사분석_프롬프트.md |
| 3 | 02_조사분석.md + 시장환경 | 판단_마스터.md | 03_판단_마스터_프롬프트.md |
| 4 | 변경 전후 비교 | 변경사항_로그.md | 04_변경사항_로그_프롬프트.md |

### 거시 파이프라인 (시장환경)

| 단계 | 입력 | 출력 | 사용 프롬프트 |
|------|------|------|--------------|
| 1 | 01_원본.md | 주제별 요약 갱신 | 06_시장환경_정리_프롬프트.md |
| 2 | 주제별 요약 전체 | 시장환경_마스터.md | 06_시장환경_정리_프롬프트.md |
| 3 | 시장환경_마스터.md | 매수시점_대출방식_판단.md | 07_매수시점_대출방식_판단_프롬프트.md |

### 비교

| 단계 | 입력 | 출력 | 사용 프롬프트 |
|------|------|------|--------------|
| 1 | 복수 매물 판단_마스터 | 비교분석.md | 05_매물비교_프롬프트.md |

## 프롬프트 사용 예시

### 조사 정리 (단건)
```
M00001_강남역오피스텔의 2026-02-17_시세조사 원본을 01_정보정리_프롬프트.md 기준으로 02_정리.md 생성해줘
```

### 조사분석 갱신
```
M00001_강남역오피스텔의 전체 조사를 반영해서 02_조사분석.md 갱신해줘
```

### 판단 마스터 갱신
```
M00001_강남역오피스텔의 판단_마스터.md 갱신해줘
```

### 전체 자동 반영 (단건 조사 추가 후)
```
M00001_강남역오피스텔에 2026-02-20_현장방문 추가했으므로 전체 업데이트 해줘
```

### 시장환경 갱신
```
30_시장환경에 2026-02-17_기준금리인하 추가했으므로 시장환경 전체 갱신해줘
```

### 매수 시점 및 대출 방식 판단 갱신
```
매수시점_대출방식_판단.md 갱신해줘
```

### 매물 비교
```
M00001과 M00002를 비교 분석해줘
```

---

## 주의사항

- 각 단계는 반드시 사람이 검토 후 저장합니다.
- 01_원본.md는 절대 수정하지 않습니다.
- AI 분석은 참고 자료이며, 투자 판단의 최종 책임은 본인에게 있습니다.
- 프롬프트를 변경하고 싶으면 충분히 테스트 후 적용합니다.
- 시장환경 정보는 빠르게 변하므로 갱신일을 항상 확인합니다.
